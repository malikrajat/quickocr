Privacy Policy

Last Updated: November 15, 2025

Overview

OCR Selection is a privacy-first Chrome extension that extracts text from screen selections using Optical Character Recognition (OCR). We are committed to protecting your privacy and ensuring transparency about data handling.

Privacy-First Promise: By default, all OCR processing happens locally on your device. No images or text are sent to external servers. No tracking or analytics.

Data Collection and Processing

Local Processing (Default)

By default, all OCR processing happens locally on your device:

  • Screen captures are processed entirely in your browser using WebAssembly (Tesseract.js)
  • No images or text are sent to external servers
  • No data leaves your device
  • No tracking or analytics

What We Store Locally

The extension stores minimal data in Chrome's local storage:

  • User Settings: OCR language preference, engine mode (fast/accurate), confidence threshold
  • No Images: We do NOT store captured images by default
  • No OCR Results: Text extraction results are not persisted

Optional Cloud OCR (Opt-In Only)

If you explicitly enable cloud OCR for higher accuracy:

  • You will see a consent dialog explaining what data is sent
  • Only the selected image region is uploaded (not full screen)
  • Images are processed and immediately deleted
  • No images are retained after processing
  • You can disable this feature at any time

Permissions Explained

The extension requests these permissions:

  • activeTab: To capture the visible browser tab when you make a selection
  • scripting: To inject the selection overlay when you activate OCR
  • storage: To save your preferences (language, settings)
  • clipboardWrite: To copy extracted text to clipboard
  • contextMenus: To add "OCR Selection" to right-click menu

We do NOT request:

  • Access to all websites
  • Access to browsing history
  • Access to personal data
  • Network access (unless cloud OCR is enabled)

Third-Party Services

  • Tesseract.js: Open-source OCR engine running locally in your browser
  • No Analytics: We do not use Google Analytics or any tracking services
  • No Ads: This extension is ad-free

Data Retention

  • Settings are stored until you uninstall the extension
  • No images or OCR results are retained
  • You can clear all data via extension settings

Your Rights

You have the right to:

  • Use the extension entirely offline (default mode)
  • Delete all stored data at any time
  • Opt out of cloud OCR (it's opt-in only)
  • Request information about data practices

Security

  • Content Security Policy enforced on all extension pages
  • No remote code execution
  • All OCR output sanitized before display
  • Secure message passing between components

Changes to This Policy

We will notify users of any material changes to this privacy policy through the extension update notes.

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • GDPR (General Data Protection Regulation)
  • CCPA (California Consumer Privacy Act)
